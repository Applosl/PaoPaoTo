## PaoPaoTo Framework

```
root/ - src # 源码目录
        / - kernel # 内核
        / - init.php # 初始化方法
    / - test # 测试代码目录
        / - App # 测试服务器 应用目录
        / - Request # Request模块测试用例目录
        / - Route # Route模块测试用例目录
        / - bootstrap.php # 测试初始化包文件
        / - index.php # 测试服务器入口文件
        / - phpunit.xml # PHPUnit 测试基境配置i 文件
        / - TestServerForNginx.conf # Nginx测试服务器配置模板
    / - vendor # composer包目录
    / - .gitignore
    / - composer.json
    / - LICENSE
    / - README.md
```
